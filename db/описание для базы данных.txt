BD
таблицы
AERODROMI - сами объекты аэродромов поиск по полям namei+namei2
VPP - таблица привязана к aerodromi по id_arpt, ВПП может быть несколько, например аэродромы Москва Внуково, Домодедово, Шереметьево
torezh таблица привязана к таблице VPP по id_vpp - отсюда берем высоты порогов из полей H_threshold, они пока в метрах.
PREPARPT - сами препятствия привязанные к AERODORMI по полю CODA. здесь поле высот Habs (тоже в метрах).
MARKER Освещение по полю Marker (значение есть "1"/нет),
LAT LON - широта долгота препятствия в градусах минутах секундах и сотых секунд
art_nat - поле "1"искусств (сделанное человеком)/ "2" - естественное, если нет данных то по умолчанию искусственное.

Сценарий на ошибки например:
Подключение к базе данных!
Нет высоты аэродрома в поле Habs!
Нет ВПП у данного аэродрома!
Нет торцов у данной ВПП!
Нет препятствий у данного аэродрома!
Нет обязательных координат в полях LAT LON препятствий.
Нет высот в поле Habs препятствия!
По закрытию программы отсоединение от базы данных!

После завершения создания файлов препятствий TXT, дать отчет список названий созданных файлов и красным выделить если файл создан "пустым".
